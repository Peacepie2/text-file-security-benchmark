# Comparative-Analysis-Encryption-Algorithms
Comparative Analysis of Some Encryption and Decryption Algorithms for Text File Security

## Author
Iniobong Peace Effiong 

## Description

This project evaluates and compares:

- AES-256-GCM
- RSA-2048 Hybrid
- ChaCha20-Poly1305
- ECC-256-ECIES

using text files of different sizes.

## Technologies Used

- Python
- Cryptography Library
- Pandas
- NumPy

## Repository Purpose

This repository contains the complete implementation and benchmarking framework used in the research project.
